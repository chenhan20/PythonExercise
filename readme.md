## steve的python練習專案 


### 建議Vscode要安裝python 套件(Python、python for VSCode)


### 需先安裝的套件如下


* [requests](https://pypi.org/project/requests/2.7.0/)
* [pandas](https://pandas.pydata.org/) 
* [beautifulsoup4]()
* [lxml]()

* [twstock]()

```
    pip install requests
    pip install pandas
    pip install lxml
```
